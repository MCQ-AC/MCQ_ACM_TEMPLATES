题意:火车从始发站（称为第1站）开出，在始发站上车的人数为a，然后到达第2站，在第2站有人上、下车，但上、下车的人数相同，因此在第2站开出是（即在到达第3 站之前）车上的人数保持为a人。从第3站起（包括第3站）上、下车的人数有一定的规律：上车的人数都是前两站上车人数之和，而下车人数等于上一站上车人数，一直到终点站的前一站（第n-1站），都满足此规律。现给出的条件是：共有N个车站，始发站上车的人数为a，最后一站下车的人数是m（全部下车）。
思路:先手动打个表，设第二站上车为a，下车为b，找到每一次总人数的关系。
#include<bits/stdc++.h>
using namespace std;
int x_a[30]={0,1,1,2,2},x_b[30]={0,0,0,0,1};
int main()
{
    std::ios::sync_with_stdio(false);
    int n,m,x,a,b;
    cin>>a>>n>>m>>x;
    for(int i=5;i<n;i++)
    {
        x_a[i]=x_a[i-1]+x_a[i-2]-1;
        x_b[i]=x_b[i-1]+x_b[i-2]+1;
    }
    b=(m-a*x_a[n-1])/x_b[n-1];
    cout<<a*x_a[x]+b*x_b[x]<<endl;
    return 0;
}
